# Adeel At Work — Live WordPress Capability Showroom

This is the upgraded portfolio source based on the uploaded `adeelatwork` design.

## What changed in this version

- Upgraded the Home hero into a **Live WordPress Capability Showroom**
- Preserved the original design direction: Satoshi typography, soft cream background, floating gradients, pill nav, project strip, portrait card, and GSAP motion
- Added a working dark/light mode toggle in the header
- Added hero proof cards: 3+ years, 50+ projects, 20+ Figma builds, 6s → 1.8s speed result
- Added a floating capability dashboard for dark/light mode, booking plugin, WooCommerce logic, and translation demos
- Added mobile responsive styles for the new hero elements
- Updated metadata for the new positioning

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Build static export

```bash
npm run build
```

The static export will be generated in the `out` folder.
